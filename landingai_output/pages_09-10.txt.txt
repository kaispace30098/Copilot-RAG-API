## A-F Life Cycle

The visual below provides an overview of important tasks in the A-F process. See the [Important Timelines](#) section for specific dates. **Start dates are in black, due dates in red.** Tasks that affect the following school year's accountability are followed by "SY 24-25".

## Arizona Department of Education Timeline

This figure provides a detailed timeline of key activities and deadlines for the Arizona Department of Education across different seasons of the year.

### Summer: May - July

- **May**
  - Self-Reported Data Submission opens (CCRI, On-Track to Grad, Credits Earned)
  - Begin Test Records Correction in ADE Connect
  - Begin graduation and cohort data validation (SY 24-25)

- **June**
  - Begin Static File V1 validation
  - Test Records Correction in ADE Connect due
  - DRP Applications due (SY 24-25)
  - Finalize graduation and cohort data validation (SY 24-25)

- **July**
  - Static File V1 Validation due
  - Self-Reported Data due (CCRI, OTG, Credits Earned)

### Fall: August - October

- **August**
  - Begin Static File V2 validation

- **September**
  - Static File V2 Validation due

- **October**
  - Begin AzEDS Income Eligibility data validation (SY 24-25)

### Winter: November - January

- **November**
  - A-F Public File release
  - A-F Appeals Window typically takes place

- **December**
  - No deadlines

- **January**
  - AzEDS Income Eligibility data validation due (SY 24-25)

### Spring: February - April

- **February**
  - Alternative Schools Application opens (SY 24-25)
  - Initial On-Track to Graduate Submission opens

- **March**
  - Initial On-Track to Graduate Submission due
  - Alternative Schools Application due (SY 24-25)

- **April**
  - No deadlines

### Visual Description

The figure is a circular timeline divided into four sections, each representing a season: Summer, Fall, Winter, and Spring. Each section lists specific tasks and deadlines relevant to that time period. The center of the circle features the Arizona Department of Education logo. The timeline is color-coded: Summer is in orange, Fall in red, Winter in blue, and Spring in green. Each task and deadline is clearly marked with the corresponding month.
